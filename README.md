# 香港臨時政府備份文件

本處備份香港臨時政府之相關宣言與文件。

有關香港臨時政府宣言，請見[維基百科條目](https://zh.wikipedia.org/zh-hk/香港臨時政府宣言)。

**注意：美國共和黨海外分部代表俞懷松（Solomon Yue）5日[表示](http://www.rfi.fr/cn/%E4%B8%AD%E5%9B%BD/20191006-%E7%BE%8E%E4%BA%BA%E8%AD%A6%E5%91%8A%E9%A6%99%E6%B8%AF%E4%B8%B4%E6%97%B6%E6%94%BF%E5%BA%9C%E5%AE%A3%E8%A8%80%E6%88%96%E9%81%AD%E5%AE%98%E6%96%B9%E6%B8%97%E9%80%8F-%E7%96%91%E4%B8%BA%E5%86%9B%E9%98%9F%E9%95%87%E5%8E%8B%E6%8F%90%E4%BE%9B%E5%80%9F%E5%8F%A3)，這（香港臨時政府宣言）很有可能是被中國的假示威者滲透，說出的假宣言，如果港民當真，可能會給北京當局出動解放軍的理由。**
